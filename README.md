# Kosmos-2-Patch14-224 Interface

This repo is a simple object detection testing platform using Kosmos-2 from Microsoft

![kosmos](assets/thumbnail.png)

## Install

```bash
git clone
cd
pip install -r requirements.txt
chmod +x run.sh
```

```bash
./run.sh
```
